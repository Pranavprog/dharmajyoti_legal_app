# **App Name**: LegaliAI

## Core Features:

- Document Summarization: Summarize uploaded legal documents into clear, plain language, preserving the original structure.
- Document Type Detection: Identify the type of document (contract, agreement, etc.) and state its purpose in one sentence.
- Interactive Legal Guidance: In chatbot mode, ask step-by-step questions to understand the legal problem and guide the user.
- Legal Assessment & Explanation: For any clause or concern, provide explanations in both legal and layman's terms. LLM acts as tool to evaluate risk and present suggestions
- Pros, Cons & Consequences: Identify pros and cons for actions (e.g., signing a contract) and list possible consequences.
- Document Upload & Display: Allow users to upload documents (PDF, DOC, image with OCR) and display them in a structured format.
- Interactive Chat Interface: A user-friendly chat interface for interactive guidance and question answering.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to convey trust and clarity.
- Background color: Light gray (#F0F2F5) for a clean and professional look.
- Accent color: Muted green (#82E0AA) for highlighting important information and calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable appearance.
- Use clear and concise icons to represent document types, actions, and warnings.
- Maintain a clean and structured layout to present information clearly.
- Subtle animations for loading indicators and interactive elements to enhance user experience.
